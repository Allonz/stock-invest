Stock Invest API v3

\u6700\u540e\u66f4\u65b0: 2026-05-16

API 概览

Stock Invest 是一个股票数据分析系统，提供 K 线数据查询、股票筛选扫描、自定义扫描、以及老虎证券自选股数据导入等功能。

基本信息

项目值
基础地址http://localhost:8090
数据格式JSON (application/json)
字符编码UTF-8

统一响应格式

所有 API 端点返回统一的 ApiResponse<T> 包装结构：

复制{
  "success": true,
  "message": null,
  "data": { ... },
  "errorType": null,
  "timestamp": "2026-05-16T12:00:00Z"
}
字段类型说明
successboolean请求是否成功
messageString | null提示信息
dataT | null响应数据（失败时为 null）
errorTypeString | null错误类型（仅失败时出现）
timestampString (ISO-8601)响应时间戳

认证

部分接口（数据导入、基于快照的筛选运行）支持可选的 API 密钥认证。在 application.yml 中配置 ingest.api-key，当配置值非空时，对应接口需要请求头 X-INGEST-API-KEY 与配置值一致，否则返回 401 Unauthorized。

1. 股票数据 API

前缀：/api/tiger | 控制器：StockController

提供股票日线 K 线数据查询，数据来源由优先级路由决定（Tiger → yfinance → TwelveData → Tiingo）。

GET/api/tiger/kline/daily/{symbol}

获取指定股票的日线 K 线数据（JSON 字符串格式）。

路径参数

参数类型必填说明
symbolString必填股票代码，例如 AAPL

cURL 示例

复制curl -X GET "http://localhost:8090/api/tiger/kline/daily/AAPL"

响应

返回 ApiResponse<String>，其中 data 为包含 K 线数据的 JSON 字符串。

复制{
  "success": true,
  "data": "[{\"symbol\":\"AAPL\",\"time\":1715644800000,...}]",
  "timestamp": "2026-05-16T12:00:00Z"
}

GET/api/tiger/kline/daily/object/{symbol}

获取指定股票的日线 K 线数据（对象格式）。 data 字段中的结构参见 KLineData 模型。

路径参数

参数类型必填说明
symbolString必填股票代码，例如 AAPL

cURL 示例

复制curl -X GET "http://localhost:8090/api/tiger/kline/daily/object/AAPL"

响应

返回 ApiResponse<KLineData>。

GET/api/tiger/kline/daily/query?symbol=

使用查询参数获取日线 K 线数据（与路径参数版同等）。

查询参数

参数类型必填说明
symbolString必填股票代码，例如 AAPL

cURL 示例

复制curl -X GET "http://localhost:8090/api/tiger/kline/daily/query?symbol=AAPL"

响应

返回 ApiResponse<String>，与路径参数版格式相同。

2. 股票扫描 API

前缀：/api/scanner | 控制器：StockScannerController

POST/api/scanner/custom

自定义股票扫描，根据市场、价格等条件筛选股票。

请求体 (JSON)

字段类型必填默认值说明
marketString否"US"市场类型，可选值：US, HK, CN, SG
categoryString否"TOP_VOLUME"分类（当前未在扫描逻辑中使用）
limitInteger否20返回股票数量上限
minPriceDouble否null最低价格筛选
maxPriceDouble否null最高价格筛选
exchangeString否null交易所（当前未在扫描逻辑中使用）

请求示例

复制{
  "market": "US",
  "category": "TOP_VOLUME",
  "limit": 20,
  "minPrice": 10.0,
  "maxPrice": 500.0,
  "exchange": null
}

cURL 示例

复制curl -X POST "http://localhost:8090/api/scanner/custom" \
  -H "Content-Type: application/json" \
  -d '{"market":"US","limit":20}'

响应

返回 ApiResponse<List<String>>，其中 data 为股票代码列表。

复制{
  "success": true,
  "data": ["AAPL","MSFT","GOOGL","AMZN","NVDA"],
  "timestamp": "2026-05-16T12:00:00Z"
}

3. 筛选查询 API

前缀：/api/screener | 控制器：ScreeningQueryController

POST/api/screener/run

执行日常筛选任务，分析指定交易日的股票数据并返回筛选结果。

查询参数

参数类型必填默认值说明
dateString否当前日期交易日期，格式：yyyy-MM-dd
limitInteger否20处理股票数量上限
windowDaysInteger否7观察窗口天数，用于趋势分析

cURL 示例

复制curl -X POST "http://localhost:8090/api/screener/run?date=2026-05-16&limit=20&windowDays=7"

响应

返回 ApiResponse<ScreenerRunResponseDto>。

复制{
  "success": true,
  "data": {
    "batchId": "SCR-20260516-XXXX",
    "tradeDate": "2026-05-16",
    "totalCandidates": 100,
    "processedStocks": 80,
    "matchedStocks": 5
  },
  "timestamp": "2026-05-16T12:00:00Z"
}

POST/api/screener/run-from-snapshot

基于快照数据执行日常筛选任务。支持可选的 API 密钥认证（X-INGEST-API-KEY）。

请求头

头类型必填说明
X-INGEST-API-KEYString可选当 ingest.api-key 配置非空时必须提供

查询参数

参数类型必填默认值说明
dateString否当前日期交易日期，格式：yyyy-MM-dd
limitInteger否20处理股票数量上限
windowDaysInteger否7观察窗口天数

cURL 示例

复制curl -X POST "http://localhost:8090/api/screener/run-from-snapshot?date=2026-05-16&limit=20&windowDays=7" \
  -H "X-INGEST-API-KEY: your-api-key"

响应

返回 ApiResponse<ScreenerRunResponseDto>，格式与 run 接口相同。

GET/api/screener/results?tradeDate=&minPrice=&maxPrice=

根据交易日期查询筛选结果。

查询参数

参数类型必填说明
tradeDateString必填交易日期，格式：yyyy-MM-dd
minPriceDouble否最低价格筛选
maxPriceDouble否最高价格筛选

cURL 示例

复制curl "http://localhost:8090/api/screener/results?tradeDate=2026-05-16&minPrice=10&maxPrice=500"

响应

返回 ApiResponse<List<ScreeningResultDto>>。

复制{
  "success": true,
  "data": [
    {
      "symbol": "AAPL",
      "price": 178.5,
      "rise": true,
      "source": "yfinance",
      "tradeDate": "2026-05-16"
    }
  ],
  "timestamp": "2026-05-16T12:00:00Z"
}

GET/api/screener/results/latest?minPrice=&maxPrice=

查询最新的筛选结果。

查询参数

参数类型必填说明
minPriceDouble否最低价格筛选
maxPriceDouble否最高价格筛选

cURL 示例

复制curl "http://localhost:8090/api/screener/results/latest?minPrice=10&maxPrice=500"

响应

返回 ApiResponse<List<ScreeningResultDto>>，与 results 接口格式相同。

4. 数据导入 API

前缀：/api/ingest | 控制器：TigerWatchlistIngestController

POST/api/ingest/tiger-watchlist

导入老虎证券自选股列表数据。支持可选的 API 密钥认证（X-INGEST-API-KEY）。

请求头

头类型必填说明
Content-TypeString必填application/json
X-INGEST-API-KEYString可选当 ingest.api-key 配置非空时必须提供

请求体 (JSON)

字段类型必填说明
tradeDateString否交易日期，格式：yyyy-MM-dd
rowsArray<TigerWatchlistRowDto>否自选股列表

其中 rows 中的每个元素（TigerWatchlistRowDto）：

字段类型说明
symbol (code)String股票代码，支持 @JsonAlias "code"
nameString股票名称
lastPriceDouble最新价格
volumeObject成交量

请求示例

复制{
  "tradeDate": "2026-05-16",
  "rows": [
    {
      "symbol": "AAPL",
      "name": "Apple Inc.",
      "lastPrice": 178.50,
      "volume": 50000000
    }
  ]
}

cURL 示例

复制curl -X POST "http://localhost:8090/api/ingest/tiger-watchlist" \
  -H "Content-Type: application/json" \
  -H "X-INGEST-API-KEY: your-api-key" \
  -d '{"tradeDate":"2026-05-16","rows":[{"symbol":"AAPL","name":"Apple Inc.","lastPrice":178.50,"volume":50000000}]}'

响应

返回 ApiResponse<TigerWatchlistIngestResponseDto>。验证失败时返回 400，带 VALIDATION_ERROR 错误类型。

复制{
  "success": true,
  "data": {
    "batchId": "INGEST-20260516-XXXX",
    "tradeDate": "2026-05-16",
    "imported": 1,
    "skipped": 0,
    "skipReasons": []
  },
  "timestamp": "2026-05-16T12:00:00Z"
}

6. 数据模型

ApiResponse<T>

统一响应包装类，所有 API 端点均返回此类型。

字段类型说明
successboolean请求是否成功
messageString | null提示信息
dataT | null响应数据
errorTypeString | null错误类型
timestampInstant时间戳 (ISO-8601)

@JsonInclude(Include.NON_NULL) — null 字段会被忽略。

CustomScanRequest

自定义扫描请求 DTO。

字段类型默认值说明
marketString"US"市场 (US|HK|CN|SG)
categoryString"TOP_VOLUME"分类（当前未使用）
limitInteger20返回数量上限
minPriceDoublenull最低价格
maxPriceDoublenull最高价格
exchangeStringnull交易所（当前未使用）

注：缺省值在构造函数中设置，当字段为 null 或空时生效。

ScreenerRunResponseDto

筛选任务执行结果 DTO。

字段类型说明
batchIdString批次标识，不可为空
tradeDateLocalDate交易日期，不可为 null
totalCandidatesInteger总候选股票数
processedStocksInteger已处理股票数
matchedStocksInteger匹配股票数

ScreeningResultDto

筛选结果记录。

字段类型说明
symbolString股票代码，不可为空
priceDouble股票价格
riseBoolean是否上涨
sourceString数据来源
tradeDateLocalDate交易日期，不可为 null

SnapshotGridViewDto

股票快照网格视图。

字段类型说明
dateHeadersList<String>日期标题列表
rowsList<SnapshotGridRowDto>行数据列表

SnapshotGridRowDto

字段类型说明
symbolString股票代码，不可为空
stockNameString股票名称
cellsList<String>各日期对应的数据单元格

TigerWatchlistIngestRequestDto

老虎自选股导入请求 DTO。

字段类型说明
tradeDateString交易日期 (yyyy-MM-dd)
rowsList<TigerWatchlistRowDto> | null自选股列表

TigerWatchlistIngestResponseDto

老虎自选股导入响应 DTO。

字段类型说明
batchIdString批次标识
tradeDateLocalDate交易日期
importedint导入数量
skippedint跳过数量
skipReasonsList<String>跳过原因列表

支持链式调用：withBatchId(), withTradeDate(), withImported(), withSkipped(), withSkipReasons()。

TigerWatchlistRowDto

自选股单行数据。

字段类型说明
symbolString股票代码，支持别名 "code"
nameString股票名称
lastPriceDouble最新价格
volumeObject成交量

@JsonAlias("code") — symbol 字段可接受名为 "code" 的 JSON 属性。

KLineData

K 线数据模型。

字段类型说明
symbolString股票代码
timelong时间戳
opendouble开盘价
highdouble最高价
lowdouble最低价
closedouble收盘价
volumelong成交量
amountdouble成交金额
itemsList<KLineIterator>K 线数据列表

KLineIterator

K 线单条数据记录。

字段类型说明
symbolString股票代码
timelong时间戳
timeStringString时间字符串格式
opendouble开盘价
highdouble最高价
lowdouble最低价
closedouble收盘价
volumelong成交量
